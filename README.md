# DLAnalytics

An abstract Analytics Framework supports:

- Unify Analytics.
- Modularize, Centralize Analytics.
- Plugin the new Analytics easier.
- Standadize tracking method and event.

## Requirements

- Xcode 11+
- Swift 5.0+
